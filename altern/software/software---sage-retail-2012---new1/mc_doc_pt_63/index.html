Sage Easy é um Modelo de Comercialização inovador que surge em alternativa ao modelo tradicional de aquisição de software e serviços pela Sage Portugal. Este Modelo permite ao cliente utilizar as Soluções Sage sem ter de as adquirir de forma definitiva.
